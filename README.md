# gRNA On-Target Efficiency Predictor

Predicts CRISPR Cas9 cleavage efficiency from a 30-nucleotide sequence.

## Live Demo
[Try it here](https://huggingface.co/spaces/birkirorri/grna-efficiency-predictor) (still WIP)

## Stack
- PyTorch CNN
- FastAPI — REST API
- Docker
- Render — API deployment
- Streamlit — frontend
- HuggingFace Spaces — UI deployment
